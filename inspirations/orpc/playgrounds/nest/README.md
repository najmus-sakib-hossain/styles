# ORPC Playground

This is a playground for [oRPC](https://orpc.unnoq.com) and [NestJS](https://nestjs.com).

## Getting Started

First, run the development server:

```bash
npm run start:dev
```

Open [http://localhost:3000](http://localhost:3000) to see the Scalar API Client.
